# 成绩计算脚本
> 浙江大学工程师学院教学管理办公室

## 使用方法
1. 将所有的个人成绩单放入`file`文件夹下
2. 运行`src/main.py`
3. 稍等即可在生成的`统计结果.xlsx`中查看

## 计算方法
（单门课程成绩 x 该门课程学分）/ 参与计算的总学分

*可以在配置里修改计算方式和参与计算的课程条件*
